<h1>Online Order Delivery Status Form</h1><br>
<h1>📖 Project Overview</h1><br>
The Online Order Delivery Status Form is a simple web application designed to collect and manage basic order delivery information from users.<br>
This project demonstrates the use of HTML form elements and JavaScript integration to capture user input such as customer name, city name, and payment status.<br>
It is an ideal beginner-level project for students who want to understand how web forms work and how JavaScript functions can be triggered from HTML events.<br>
<h1>🎯 Objectives</h1>
The main objectives of this project are:<br>
<li>To understand the structure of an HTML web page.</li>
<li>To learn how to create and design HTML forms.</li>
<li>To collect user input using form fields.</li>
<li>To connect HTML with JavaScript for interactive behavior.</li>
<li>To demonstrate basic client-side functionality in web development.</li>
<h1>🛠️ Technologies Used</h1>
<li>HTML5 – For creating the structure of the web page.</li>
<li>JavaScript – For handling form submission and functionality.</li>
<li>Basic Web Development Concepts</li>

<h1>🧑‍💻 How It Works</h1>
<b>1.</b>The user opens the web page.<br>
<b>2.</b>The user enters the following details:<br>
<li>Customer Name</li>
<li>City Name</li>
<li>Payment Status</li>
<b>3.</b>After entering the information, the user clicks the Submit button.<br>
<b>4.</b>The button triggers the JavaScript function (getOrder()) which processes the form data.<br>
